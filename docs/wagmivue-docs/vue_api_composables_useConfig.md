# useConfig ​
Composable for getting `Config` from the `WagmiPlugin`.
## Import ​
ts```
import { useConfig } from 'wagmi'
```

## Usage ​
index.vue
vue```
<script setup lang="ts">
import { useConfig } from 'wagmi'
const config = useConfig()
</script>
```

## Return Type ​
ts```
import { type UseConfigReturnType } from 'wagmi'
```
